## Installation Commands

### Basic Installation
```bash
pip install -r requirements.txt
```

### GPU Support (CUDA 11.8)
```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
```

### Development Installation
```bash
pip install -r requirements.txt
pip install -r requirements-dev.txt  # If created
```

## Version Constraints

- Python: >=3.8, <3.12
- CUDA: 11.8 or 12.1 (for GPU training)
- Memory: >=32GB RAM (for training)
- Storage: >=256GB SSD (for datasets)

## Platform Support

- Linux (Ubuntu 20.04+)
- macOS (Intel/Apple Silicon)
- Windows (with WSL2 recommended)

## Notes

- Use virtual environment to avoid conflicts
- Install CUDA toolkit separately for GPU support
- Some packages may require compilation (e.g., flash-attn)
- Consider using conda for easier dependency management
